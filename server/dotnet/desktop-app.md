### 跨平台应用
|名称|网址|说明|
|------|------|------|
|Xwt|https://github.com/mono/xwt|Xwt 是为 Mono 和 .NET 创建的基于 .NET 框架的跨平台开源 UI 工具包，它使用通用的 API，构建于所支持的每种平台的原生 Widget 工具包之上。它主要用来创建基于同一个代码库的跨平台桌面应用程序。使用 Xwt 创建的应用程序在相应平台上具有本地化的视觉效果|
